Summary:
Puck 0.19 introduces the Slots API (successor to DropZones) for nesting components using slot fields, major performance improvements, metadata API, walkTree utility, selectors for usePuck, useGetPuck hook, new react-router recipe, and various field improvements (placeholders, step size, hiding fields). Includes upgrade guidance and changelog.

Key features documented:
- Slots API (slot field, defaultProps, resolveData usage)
- walkTree function to traverse nested slots and manipulate nodes
- createUsePuck selectors and useGetPuck hook
- Metadata API for injecting data into components
- New react-router recipe
- Performance improvements and getters for item selectors
- resolveData trigger param
- Custom label icons and field placeholders
- Step size for number fields and hiding fields
- RootConfig type and replaceRoot action
